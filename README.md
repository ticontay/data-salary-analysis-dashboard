# Data Salary Analysis Dashboard

Live demo: https://data-salary-analysis-dashboard.streamlit.app

Interactive dashboard built with Python to explore salary data in data-related roles. 
The analysis focuses on job title, experience level, contract type, company size, work type, and location.

## Stack
- Python
- Pandas
- Streamlit
- Plotly

## Features
- Interactive filters (year, seniority, contract type, company size)
- Salary KPIs (average, maximum, total records)
- Bar chart with top 10 roles by average salary
- Salary distribution histogram
- Work type proportion (remote, hybrid, on-site)
- World map showing average Data Scientist salaries by country

## Data Source
CSV file from a public GitHub repository (originally sourced from Kaggle):
https://raw.githubusercontent.com/vqrca/dashboard_salarios_dados/refs/heads/main/dados-imersao-final.csv
